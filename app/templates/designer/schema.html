Export

Download the whole configuration as JSON — tables, fields, relations, forms, menus, shared dashboards, workflows, triggers, roles & permissions, integrations (connections, feeds, webhooks & pull sources) and auto-number counters. Schema only — not the data rows.

An import does not restore connection tokens, webhook tokens/secrets (rotate a webhook to get a fresh receive URL), pull-source headers/auth secrets (re-enter them) or uploaded files (those are data, not schema). It does include data-source passwords (needed to recreate tables in other databases) — treat the export file as a secret.

Import
